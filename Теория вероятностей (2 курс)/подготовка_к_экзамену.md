# Подготовка к экзамену по теории вероятностей

Справочник теории, терминов и формул. Пополняется по мере разбора билетов.

---

## 1. Базовые понятия

| Термин | Определение |
|--------|-------------|
| **Пространство элементарных исходов** $\Omega$ | Множество всех возможных исходов эксперимента |
| **Событие** $A$ | Подмножество $\Omega$ |
| **Вероятность** $P(A)$ | Числовая мера возможности наступления события, $0 \le P(A) \le 1$ |
| **Достоверное событие** | $\Omega$, $P(\Omega)=1$ |
| **Невозможное событие** | $\varnothing$, $P(\varnothing)=0$ |
| **Противоположное событие** $\bar A$ | $\Omega \setminus A$, $P(\bar A)=1-P(A)$ |

### Аксиомы Колмогорова
1. $P(A) \ge 0$
2. $P(\Omega) = 1$
3. Для попарно несовместимых $A_1, A_2, \ldots$: $P\!\left(\bigcup_i A_i\right) = \sum_i P(A_i)$

---

## 2. Операции над событиями и формулы

| Формула | Условие | Смысл |
|---------|---------|-------|
| $P(A \cup B) = P(A) + P(B) - P(AB)$ | всегда | Объединение |
| $P(A \cup B) = P(A) + P(B)$ | $AB = \varnothing$ | Несовместные события |
| $P(\bar A) = 1 - P(A)$ | всегда | Противоположное |
| $P(A \mid B) = \dfrac{P(AB)}{P(B)}$ | $P(B)>0$ | Условная вероятность |
| $P(AB) = P(A)\,P(B)$ | независимость | Произведение |
| $P(AB) = P(B)\,P(A\mid B)$ | всегда | Теорема умножения |

### Формула полной вероятности
Если $H_1,\ldots,H_n$ — полная группа гипотез ($H_i H_j = \varnothing$, $\bigcup H_i = \Omega$), то:
$$P(A) = \sum_{i=1}^{n} P(H_i)\,P(A \mid H_i)$$

### Формула Байеса
$$P(H_k \mid A) = \frac{P(H_k)\,P(A \mid H_k)}{\sum_{i=1}^{n} P(H_i)\,P(A \mid H_i)}$$

## 3. Комбинаторика

| Формула | Когда использовать |
|---------|-------------------|
| $P_n = n!$ | Перестановки $n$ различных элементов |
| $A_n^k = \dfrac{n!}{(n-k)!}$ | Размещения без повторений |
| $C_n^k = \binom{n}{k} = \dfrac{n!}{k!(n-k)!}$ | Сочетания без повторений |

**Классическая схема:** $P(A) = \dfrac{m}{n}$, где $m$ — число благоприятных исходов, $n$ — число равновозможных исходов.

---

## 4. Схема Бернулли и биномиальное распределение

$n$ **независимых** испытаний, в каждом «успех» с вероятностью $p$.

$$P_n(k) = C_n^k\, p^k (1-p)^{n-k}, \quad k = 0,1,\ldots,n$$

| Характеристика | Формула |
|----------------|---------|
| $M[X]$ | $np$ |
| $D[X]$ | $np(1-p)$ |
| $\sigma[X]$ | $\sqrt{np(1-p)}$ |

### Наивероятнейшее число успехов $k^*$
Найти $k$, при котором $P_n(k)$ максимально. Удобно сравнивать отношения:
$$\frac{P_n(k)}{P_n(k-1)} = \frac{n-k+1}{k}\cdot\frac{p}{1-p}$$
- если отношение $> 1$, вероятность растёт;
- $k^* = \lfloor (n+1)p \rfloor$ или соседние значения (проверить оба).

## 5. Гипергеометрическое распределение

Из $N$ объектов, среди которых $K$ «успешных», **без возвращения** выбирают $n$ объектов. Число успехов $X$:

$$P(X=k) = \frac{C_K^k\, C_{N-K}^{n-k}}{C_N^n}, \quad k = \max(0,n+K-N), \ldots, \min(n,K)$$

| Характеристика | Формула |
|----------------|---------|
| $M[X]$ | $n\dfrac{K}{N}$ |
| $D[X]$ | $n\dfrac{K}{N}\dfrac{N-K}{N}\dfrac{N-n}{N-1}$ |

---

## 6. Дискретная случайная величина (СВ)

| Понятие | Обозначение / формула |
|---------|----------------------|
| **Закон распределения** | Таблица $x_i$, $p_i$, $\sum p_i = 1$ |
| **Функция распределения** | $F(x) = P(X \le x)$ |
| **Математическое ожидание** | $M[X] = \sum x_i p_i$ |
| **Момент 2-го порядка** | $M[X^2] = \sum x_i^2 p_i$ |
| **Дисперсия** | $D[X] = M[X^2] - (M[X])^2$ |
| **Свойства** | $M[aX+b] = aM[X]+b$; $D[aX+b] = a^2 D[X]$ |

Для **независимых** $X_1,\ldots,X_n$: $M\left[\sum X_i\right] = \sum M[X_i]$, $D\left[\sum X_i\right] = \sum D[X_i]$.

---

## 7. Непрерывная случайная величина

| Понятие | Формула |
|---------|---------|
| **Плотность** $f(x)$ | $f(x) = F'(x)$ (где производная существует) |
| Нормировка | $\displaystyle\int_{-\infty}^{\infty} f(x)\,dx = 1$ |
| Вероятность попадания в промежуток | $P(a < X < b) = \int_a^b f(x)\,dx = F(b) - F(a)$ |
| **Математическое ожидание** | $\displaystyle M[X] = \int_{-\infty}^{\infty} x f(x)\,dx$ |
| **Дисперсия** | $D[X] = M[X^2] - (M[X])^2$ |

---

## 8. Нормальное распределение $N(\mu, \sigma^2)$

$$f(x) = \frac{1}{\sigma\sqrt{2\pi}}\, e^{-\frac{(x-\mu)^2}{2\sigma^2}}$$

| Свойство | Значение |
|----------|----------|
| $M[X]$ | $\mu$ (абсцисса максимума плотности) |
| $D[X]$ | $\sigma^2$ |
| Высота максимума | $f(\mu) = \dfrac{1}{\sigma\sqrt{2\pi}}$ |

**Стандартное нормальное:** $\mu=0$, $\sigma^2=1$, $M[X]=0$.

## 9. Геометрическая вероятность (задачи на встречу)

Точки $(x, y)$ равномерно в прямоугольнике $\Omega$. Тогда:
$$P(A) = \frac{S(A)}{S(\Omega)}$$

Классика: два человека приходят в интервале длины $T$, ждут по $t$ минут — область встречи в квадрате $T \times T$.

## 10. Выборочные характеристики

По вариационному ряду $(x_i, n_i)$, $n = \sum n_i$:

| Характеристика | Формула |
|----------------|---------|
| **Выборочное среднее** | $\bar{x} = \dfrac{1}{n}\sum x_i n_i$ |
| **Вариационный размах** | $R = x_{\max} - x_{\min}$ |
| **Мода** | Вариант с наибольшей частотой $n_i$ |
| **Выборочная дисперсия (несмещённая)** | $s^2 = \dfrac{1}{n-1}\sum n_i (x_i - \bar{x})^2$ |
| **Точечная оценка $M$** генеральной совокупности | $\hat{\mu} = \bar{x}$ |
| **Точечная оценка $D$** генеральной совокупности | $\hat{\sigma}^2 = s^2$ |

## 12. Независимые испытания с разными $p_i$

Если $X_i$ — индикатор $i$-го успеха, $P(X_i=1)=p_i$, испытания независимы, то для $Y = X_1 + X_2 + \ldots + X_n$:

$$M[Y] = \sum_{i=1}^{n} p_i, \qquad D[Y] = \sum_{i=1}^{n} p_i(1-p_i)$$

Закон распределения $Y$ находят перебором всех $2^n$ исходов (см. вариант 3, задача 4).
