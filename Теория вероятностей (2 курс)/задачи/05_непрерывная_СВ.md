# Задание 5. Непрерывная СВ: $F(x)$, $f(x)$, $M$, $D$

**Баллы на экзамене:** 8 · **Тип:** СКО

## Метод

$f(x)=F'(x)$; $P(a<X<b)=F(b)-F(a)$; $M[X]=\int x f(x)\,dx$; нормировка $\int f=1$.

**Лекции:** [лекция_04](../Лекции%20markdown/лекция_04.md), [лекция_05](../Лекции%20markdown/лекция_05.md)

---

## Вариант 1 · [решение](../решения/вариант_1.md#5-задание-5)

Пусть непрерывная случайная величина $X$ имеет функцию распределения

$$
F(x) = \begin{cases}
0, & \text{если } x \le 0, \\
\frac{1}{2}(x^3 - 3x^2 + 3x), & \text{если } 0 < x \le 2, \\
1, & \text{если } x > 2.
\end{cases}
$$

Найти математическое ожидание $M[X]$ и вероятность $P(0 \le X \le 1)$.

---

## Вариант 2

Пусть непрерывная случайная величина $X$ имеет плотность вероятности

$$
f(x) = \begin{cases}
0, & \text{если } x < 0,\ x > 1, \\
4x^2, & \text{если } 0 \le x \le 1.
\end{cases}
$$

Найти дисперсию $D[X]$ и вероятность $P(0{,}2 < X < 0{,}8)$.
