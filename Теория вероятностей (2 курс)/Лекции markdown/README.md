# Конспекты лекций

Markdown-версии презентаций из папки `Презентации/`.

- Вступительный слайд с контактами преподавателя **не включён**.
- Каждый файл начинается с `# Лекция N. Тема`.
- Формулы в LaTeX; пропущенные при конвертации — восстановлены вручную/скриптом.
- В конце каждой лекции — **Summary**: определения, формулы, что запомнить.

**Постобработка:** `python3 Scripts/enhance_lectures.py`
