# Лекция 3. Повторные независимые испытания

Пусть испытания многократно повторяются при одном и том же комплексе условий. Причем вероятность появления события А в каждом испытании не зависит от исходов других испытаний, то есть постоянна.

Тогда

такие испытания называют независимыми повторными испытаниями.

последовательность таких испытаний называют схемой Бернулли

Далее будем рассматривать лишь такие независимые испытания, в которых событие А имеет одну и ту же вероятность p.

---

## Повторные независимые испытания: примеры

Подбрасываем игральный кубик n раз. Выпадение числа очков от 1 до 6 происходит с вероятностью p=1/6 в каждом из испытаний;

Подбрасывается n раз монета. Выпадение орла или решки происходит с вероятностью p=½ в каждом испытании.

---

## Повторные независимые испытания

В задачах на повторные независимые испытания представляют интерес :

1) вероятность наступления события А ровно m раз в n испытаниях  Pn(m);

2) вероятность того, что число наступления события А в n повторных независимых испытаниях заключено в некоторых пределах от a до b

Pn(a≤m ≤ b).

---

## Формула Бернулли

Пусть производится n испытаний Бернулли.

Вероятность того, что в этих испытаниях событие А произойдет ровно m раз можно найти по формуле Бернулли:

n — число испытаний

p — вероятность появления события А в одном испытании

q =1-p  - вероятность не появления события А в одном испытании

Рn(m) — вероятность того, что событие А появится ровно m раз в n испытаниях

$$P_n(m) = C_n^m\, p^m q^{n-m}, \quad q = 1-p$$

> *Часть содержимого — графические формулы в PPTX.*

## Наивероятнейшее число появление события А в n повторных независимых испытаниях

Решение:

Среди 5 отобранных деталей может быть 0, 1, 2, 3, 4, 5 бракованных. Значит необходимо найти P5(0); P5(1); P5(2); P5(3); P5(4); P5(5).

P5(1)=0,4096; 		P5(4)=0,0064;

## Полученные вероятности изобразим графически точками с координатами (m, Pn(m)). Соединяя эти точки, получим многоугольник или полигон распределения вероятностей.

Рассматривая  многоугольник  распределения вероятностей мы видим, что есть такие  значения m (в данном  случае, одно  m0=1), которые обладают  наибольшей вероятностью Рn(m).

## Наивероятнейшее число появление события А в n повторных независимых испытаниях

Число m0  наступления события А в n независимых  испытаниях называется наивероятнейшим, если вероятность  осуществления этого события Рn(m0) по крайней мере не меньше  вероятностей других событий Рn(m) при любом m.

Для нахождения m0 используется двойное неравенство:

n p - q ≤ m0 ≤ np + p

или

n p - q ≤ m0 ≤ np —q+1

---

## Наивероятнейшее число появления события А в n повторных независимых испытаний

Так как наивероятнейшее число может быть только целым, то:

если границы дробные, то m0 примет только одно целое значение;

если границы целые (отличаются на 1), то m0  может принимать два значения, равные граничным. Тогда для определения наивероятнейшего числа нужно сравнить вероятности на границах.

n p - q ≤ m0 ≤ np + p

5⦁0,2-0,8 ≤ m0 ≤ 5⦁0,2+0,2

0,2≤m0≤1,2                          m0=1

Решение:

1) Событие А — 21 июля пойдет дождь.

2) По условию: p=0,3,  q=0,7,  n=30.

n∙p - q ≤ m0 ≤ n∙p + p

0,3∙30 — 0,7 ≤ m0 ≤ 0,3∙30 + 0,3

8,3 ≤ m0 ≤ 9,3

m0 = 9

Ответ:

наивероятнейшее число дождливых дней 21 июля на ближайшие 30 лет равно 9.
